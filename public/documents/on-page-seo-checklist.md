# On-Page SEO Checklist

## Checklist Halaman
- Title tag mengandung keyword utama dan value proposition
- Meta description jelas, menarik, dan memiliki CTA
- Satu H1 utama pada setiap halaman
- H2 dan H3 tersusun sesuai topik
- URL singkat, jelas, dan mengandung keyword relevan
- Konten menjawab search intent user
- Internal link menuju halaman terkait
- Image alt text deskriptif
- FAQ menjawab pertanyaan umum user
- CTA mudah ditemukan

## Output
Checklist ini membantu memastikan halaman lebih relevan untuk user dan search engine sebelum dipublikasikan.
