# Technical SEO Audit Report

## Objective
Mengecek kondisi teknis website agar halaman penting lebih mudah di-crawl, di-index, dan dipahami search engine.

## Area Audit
- Indexing dan crawlability
- Sitemap.xml dan robots.txt
- Title tag, meta description, dan heading
- Broken link dan redirect issue
- Mobile usability dan page speed
- Image alt text dan struktur internal link

## Format Temuan
| Issue | URL | Dampak | Prioritas | Rekomendasi |
|---|---|---|---|---|
| Meta description kosong | /layanan | CTR berpotensi rendah | High | Tambahkan meta description yang menjelaskan layanan dan CTA |
| H1 lebih dari satu | /blog/contoh | Struktur konten kurang jelas | Medium | Gunakan satu H1 utama dan H2 untuk subbagian |
| Gambar tanpa alt text | /produk | Aksesibilitas dan konteks gambar kurang | Medium | Tambahkan alt text deskriptif |

## Output
Audit menghasilkan daftar issue prioritas agar tim bisa memperbaiki masalah SEO secara bertahap dan terukur.
